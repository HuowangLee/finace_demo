# 账户资金流异常研判

## 所属场景

风险审查

## 项目类型

展示用虚构项目

## 甲方背景

**甲方：某城市商业银行反洗钱中心**

甲方每天产生大量反洗钱预警，包括频繁转入转出、分散转入集中转出、短期快进快出、账户行为异常、交易对手集中、疑似空壳企业交易等。人工审核压力大，低价值重复告警多，判断标准不统一。

## 甲方需求

建设账户资金流异常研判系统，对可疑交易线索进行分层、聚类和辅助分析，帮助反洗钱人员快速定位高风险线索和证据链。

## AI+金融解决方案

1. 接入账户信息、客户 KYC、交易流水、对手方账户、历史案例、黑名单、司法舆情和反洗钱规则。
2. 识别快进快出、拆分交易、循环交易、异常集中交易等模式。
3. 构建账户关系图谱，识别账户群、资金链路和关联账户。
4. 对预警进行风险分层，优先推送高风险案件。
5. 自动生成可疑交易分析摘要和人工核查建议。

## 核心功能模块

| 模块 | 说明 |
|---|---|
| 交易模式识别 | 识别快进快出、拆分交易、循环交易、异常集中交易。 |
| 账户关系图谱 | 识别账户群、资金链路和关联账户。 |
| 风险分层 | 区分高风险、中风险、低风险预警。 |
| 案件摘要生成 | 生成可疑交易分析摘要。 |
| 处置辅助 | 给出进一步核查方向和证据链。 |

## 数据来源

- 账户信息
- 客户 KYC
- 交易流水
- 对手方账户
- 历史案例
- 黑名单
- 司法舆情
- 反洗钱规则

## 关键技术

- 异常检测
- 图计算
- 关系网络分析
- 规则引擎
- 聚类分析
- 摘要生成

## 预期价值

- 降低低价值预警审核压力
- 提升高风险线索发现能力
- 统一反洗钱研判口径
- 加快可疑交易报告准备

## 页面展示摘要

面向城市商业银行反洗钱中心的账户资金流异常研判场景，通过 AI 对金融业务资料、市场数据、规则条款和历史经验进行解析、检索、建模与辅助生成，帮助业务人员提升处理效率、降低人工遗漏风险，并保留可追溯的决策依据。
