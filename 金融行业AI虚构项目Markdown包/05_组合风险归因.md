# 组合风险归因

## 所属场景

投资决策

## 项目类型

展示用虚构项目

## 甲方背景

**甲方：某银行理财子公司混合资产投资部**

甲方管理多只固收+产品，持仓包括利率债、信用债、可转债、权益基金、ETF 和现金类资产。市场波动时，产品净值回撤原因难以快速解释，投资经理、风控部门和产品经理需要快速判断亏损来源、风险暴露和对客户的解释口径。

## 甲方需求

建设组合风险归因系统，对组合收益、波动、回撤、行业暴露、因子暴露和信用主体暴露进行自动拆解，辅助投资经理形成风险复盘和调整依据。

## AI+金融解决方案

1. 接入持仓、估值、交易流水、市场行情、基准指数和风险因子数据。
2. 自动拆解组合收益来源和亏损来源。
3. 定位造成回撤的主要资产、行业、因子或个券。
4. 模拟利率上行、信用利差扩大、权益下跌等压力情景。
5. 生成归因报告和风险解释摘要，供投资经理、风控人员和产品经理使用。

## 核心功能模块

| 模块 | 说明 |
|---|---|
| 收益归因 | 拆解票息、资本利得、权益涨跌、汇率变化等收益来源。 |
| 风险归因 | 拆解利率、信用、行业、因子和个券风险。 |
| 回撤分析 | 定位主要亏损资产和触发事件。 |
| 压力测试 | 模拟利率、信用、权益、汇率等市场情景。 |
| 报告生成 | 生成组合风险解释、客户沟通摘要和内部复盘材料。 |

## 页面展示摘要

面向银行理财子公司混合资产投资部的组合风险归因场景，通过 AI 对金融业务资料、市场数据、规则条款和历史经验进行解析、检索、建模与辅助生成，帮助业务人员提升处理效率、降低人工遗漏风险，并保留可追溯的决策依据。
