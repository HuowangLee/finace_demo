# 交易指令辅助

## 所属场景

投资决策

## 项目类型

展示用虚构项目

## 甲方背景

**甲方：某资管机构多资产交易团队**

甲方投资经理每天根据策略判断和投委会决策，向交易员下达股票、债券、基金、ETF 等交易需求。正式执行前，交易员需要核对产品账户、交易标的、方向、数量、价格区间、投资比例、剩余额度、合规限制和流动性情况，人工沟通和校验成本较高。

## 甲方需求

建设交易指令辅助系统，将投资经理的自然语言交易意图转化为标准化交易指令草案，并在交易前完成基础合规校验、额度校验和流动性提示。

## AI+金融解决方案

1. 接入组合持仓、产品合同、交易规则、合规限制、市场行情和流动性数据。
2. 识别投资经理输入中的交易品种、方向、账户、数量、金额和价格约束。
3. 自动生成标准化交易指令草案，供交易员确认。
4. 在下达前校验投资范围、集中度、禁投名单、额度和流动性风险。
5. 所有指令建议必须经过人工确认，并形成完整留痕。

## 核心功能模块

| 模块 | 说明 |
|---|---|
| 交易意图识别 | 识别买入、卖出、调仓、对冲等交易意图。 |
| 指令草案生成 | 生成标准化交易指令草案和交易要素。 |
| 交易前校验 | 检查投资范围、集中度、禁投名单和剩余额度。 |
| 流动性提示 | 根据成交量、价差、持仓规模提示执行风险。 |
| 人工确认留痕 | 交易员确认后进入正式流程，系统记录全流程证据。 |

- 减少交易指令沟通成本
- 降低交易要素录入错误
- 提升交易前合规校验效率
- 保留人工确认和审计证据

## 页面展示摘要

面向资管机构多资产交易团队的交易指令辅助场景，通过 AI 对金融业务资料、市场数据、规则条款和历史经验进行解析、检索、建模与辅助生成，帮助业务人员提升处理效率、降低人工遗漏风险，并保留可追溯的决策依据。
