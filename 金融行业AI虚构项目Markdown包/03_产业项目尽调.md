# 产业项目尽调

## 所属场景

投研分析

## 项目类型

展示用虚构项目

## 甲方背景

**甲方：某大型能源集团下属投资管理公司**

甲方长期参与新能源、储能、氢能、环保材料等产业项目投资。项目立项前需要阅读商业计划书、可研报告、审计报告、行业政策、股权结构文件、合作协议和历史项目资料。资料多、格式杂、时间紧，投资经理要花大量时间做基础整理。

## 甲方需求

建设产业项目尽调辅助系统，对拟投资项目自动完成资料解析、股权穿透、财务指标抽取、产业链位置识别、风险事项识别和尽调问题生成。

## AI+金融解决方案

1. 接入项目申报材料、可研报告、商业计划书、财务报表、合同协议和行业资料。
2. 对 PDF、Word、Excel、PPT、扫描件进行统一解析和结构化处理。
3. 识别项目公司股权结构、实控人、关联方、上下游客户和供应商。
4. 抽取收入、利润、现金流、负债、投资回收期、IRR 等关键指标。
5. 生成尽调问题清单和初步风险清单，辅助投资团队判断项目是否进入下一阶段。

## 核心功能模块

| 模块 | 说明 |
|---|---|
| 资料自动解析 | 解析多格式项目资料并按章节和主题归档。 |
| 股权穿透分析 | 识别项目公司股东、实控人、关联企业和历史变更。 |
| 财务指标抽取 | 抽取收入、利润、现金流、负债、投资回收期等指标。 |
| 产业链识别 | 判断项目所处产业链环节及上下游依赖关系。 |
| 尽调问题生成 | 根据资料缺口和风险线索生成补充尽调问题。 |

## 页面展示摘要

面向大型能源集团下属投资管理公司的产业项目尽调场景，通过 AI 对金融业务资料、市场数据、规则条款和历史经验进行解析、检索、建模与辅助生成，帮助业务人员提升处理效率、降低人工遗漏风险，并保留可追溯的决策依据。
