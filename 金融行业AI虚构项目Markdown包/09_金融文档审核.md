# 金融文档审核

## 所属场景

客户服务与运营处理

## 项目类型

展示用虚构项目

## 甲方背景

**甲方：某大型银行投资银行部**

甲方在债券承销、并购融资、银团贷款、产业基金合作等业务中，需要处理大量合同、尽调报告、财务报表、担保协议、授信材料、项目说明书和审批材料。业务人员需要反复核对主体名称、金额、期限、利率、担保方式、附件、签章和授权文件。

## 甲方需求

建设金融文档审核系统，对合同、开户资料、授信材料、基金合同、尽调报告和投决材料进行字段抽取、缺失检查、一致性校验和风险提示。

## AI+金融解决方案

1. 对合同、报告、申请表、财务表格、影像材料进行解析。
2. 抽取主体、金额、期限、利率、担保方式、账户信息等关键字段。
3. 跨文档比对核心要素，发现材料之间的不一致。
4. 识别关键条款缺失、签章缺失、授权文件缺失和附件缺失。
5. 生成审核问题清单和修改建议。

## 核心功能模块

| 模块 | 说明 |
|---|---|
| 字段抽取 | 抽取主体、金额、期限、利率、担保方式、账户信息。 |
| 一致性校验 | 比对不同文档中的关键字段是否一致。 |
| 缺失检查 | 检查附件、签章、授权文件、审批材料是否齐全。 |
| 条款识别 | 识别重大违约、提前到期、担保责任等关键条款。 |
| 审核意见生成 | 输出问题清单和修改建议。 |

## 页面展示摘要

面向大型银行投资银行部的金融文档审核场景，通过 AI 对金融业务资料、市场数据、规则条款和历史经验进行解析、检索、建模与辅助生成，帮助业务人员提升处理效率、降低人工遗漏风险，并保留可追溯的决策依据。
