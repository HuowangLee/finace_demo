# 投资合规校验

## 所属场景

风险审查

## 项目类型

展示用虚构项目

## 甲方背景

**甲方：某公募基金公司合规风控部**

甲方管理股票型、债券型、混合型、指数型、专户产品等多类产品。每只产品都有不同的基金合同、投资范围、集中度限制、禁投范围、关联交易要求和内部风控规则。合规人员需要在交易前、交易中、交易后持续校验。

## 甲方需求

建设投资合规校验系统，将基金合同、监管规则和内部制度转化为可执行规则，支持交易前校验、持仓后监测和合规解释。

## AI+金融解决方案

1. 解析基金合同、投资协议、监管文件和内部制度。
2. 抽取投资范围、比例限制、禁投要求、审批要求和例外条款。
3. 将自然语言条款转化为结构化规则和校验逻辑。
4. 在交易前、交易后和日终估值环节进行自动校验。
5. 输出触发规则、来源条款和处理建议，供合规人员复核。

## 核心功能模块

| 模块 | 说明 |
|---|---|
| 合同条款解析 | 自动抽取投资范围、比例限制和禁投要求。 |
| 规则结构化 | 将自然语言条款转化为可执行校验规则。 |
| 交易前校验 | 下单前检查是否违反限制。 |
| 持仓后监测 | 每日监测持仓比例、集中度和偏离度。 |
| 合规解释 | 输出触发规则、来源条款和处理建议。 |

## 页面展示摘要

面向公募基金公司合规风控部的投资合规校验场景，通过 AI 对金融业务资料、市场数据、规则条款和历史经验进行解析、检索、建模与辅助生成，帮助业务人员提升处理效率、降低人工遗漏风险，并保留可追溯的决策依据。
