# 标的画像生成

## 所属场景

投研分析

## 项目类型

展示用虚构项目

## 甲方背景

**甲方：某公募基金权益研究部**

甲方需要持续跟踪数百家上市公司。每次新增研究标的时，研究员要手工整理主营业务、财务表现、股权结构、管理层变化、公告事件、机构调研纪要和风险事项，耗时长且格式不统一。

## 甲方需求

输入股票代码、公司名称或债券主体名称后，系统自动生成结构化标的画像，用于标的初筛、晨会讨论、研究员交接和投研归档。

## AI+金融解决方案

1. 围绕上市公司、债券发行主体、基金产品或项目公司建立统一画像模板。
2. 自动抓取并解析公告、财报、调研纪要、新闻、评级报告和内部研究材料。
3. 抽取经营指标、财务指标、估值指标、舆情事件和风险事项。
4. 生成结构化画像报告，并保留每项结论的来源依据。
5. 支持画像版本更新，记录主体信息和研究观点的变化。

## 核心功能模块

| 模块 | 说明 |
|---|---|
| 基本信息画像 | 整理主营业务、地区、行业分类、上市时间和核心产品。 |
| 财务表现分析 | 抽取收入、利润、现金流、毛利率、负债率等指标。 |
| 经营变化识别 | 识别重大合同、产能投放、业务结构变化和管理层变化。 |
| 风险事项汇总 | 汇总诉讼、处罚、评级调整、业绩下滑和负面舆情。 |
| 投资线索提炼 | 提炼增长逻辑、政策催化、行业变化和估值关注点。 |

## 数据来源

- 股票公告
- 年度报告
- 季度报告
- 机构调研纪要
- 新闻舆情
- 评级报告
- 内部研究材料
- 行情与估值数据

## 关键技术

- 信息抽取
- 财务指标解析
- 事件抽取
- 企业知识图谱
- 摘要生成
- 来源溯源

## 预期价值

- 缩短标的初筛时间
- 统一研究画像模板
- 提升跨研究员交接效率
- 帮助投委会快速理解标的情况

## 页面展示摘要

面向公募基金权益研究部的标的画像生成场景，通过 AI 对金融业务资料、市场数据、规则条款和历史经验进行解析、检索、建模与辅助生成，帮助业务人员提升处理效率、降低人工遗漏风险，并保留可追溯的决策依据。
